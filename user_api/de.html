Anmeldung - Installation - Benutzerhandbuch - ORLANDOviols

Da alle Konzerte unseres Ensembles ORLANDOviols im Jahr 2020 wegen der Covid19-Pandemie abgesagt wurden, haben wir die Gelegenheit genutzt, ein Gerät für die Fernprobe zu entwickeln: die ORLANDOviols Consort Box, kurz ovbox. Mit diesem Gerät können wir uns gegenseitig hören, als wären wir im selben Raum, indem wir eine virtuelle akustische Umgebung schaffen. Wir wollen unser Wissen mit anderen teilen und haben die Technologie der Öffentlichkeit zugänglich gemacht.

Dieses System kann von jedem kostenlos genutzt werden. Um es zu nutzen, brauchst du entweder einen Raspberry Pi Mini-Computer mit spezieller Software oder eine Desktop-Anwendung für Ubuntu Linux oder MacOS.

Das System ist für niedrige Latenzzeiten und qualitativ hochwertiges räumliches Audio optimiert. Es basiert auf dem TASCAR-Tool, das in der Hörforschung und bei unseren Konzerten eingesetzt wird.
