ovbox
ORLANDOviols consort box

Remote collaboration project: 'ovbox'

login - installation - user manual

Since all concerts of our ensemble ORLANDOviols were cancelled in 2020 due to the Covid19 pandemic, we took the opportunity to develop a remote rehearsal device: the ORLANDOviols consort box, or ovbox for short. This device allows us to hear each other as if we were in the same room. We believe in sharing our knowledge and have made the technology available to the public.

This system is free for anyone to use. To use it, you need either a Raspberry Pi mini computer with special software, or a desktop application for Ubuntu Linux or MacOS.

The system is optimised for low latency and high quality spatial audio. It is based on the TASCAR tool used in hearing research and music applications.
